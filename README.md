# Course Helper App

## Project Summary
**Course Helper App** is a desktop app built with JavaFX created to help students manage course resources and schedules, and enhanced with an AI assistant for personalized guidance. 

Features:
- Calendar creation and management
- Task list creation and management
- Deadline reminders
- Dedicated page per course
- AI-powered Chatbot to assist with studying and deadline reminders.
